== Installation ==

1. Install `what-if-bitcoin` via the plugin directory or Upload the entire `what-if-bitcoin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will find 'What If Bitcoin?' menu in your WordPress admin panel.
4. Then set your required plugin setting 
5. Just copy paste this shortcode <b>[what-if-bitcoin]</b> to any page or in the widget.

For basic usage, you can also have a look at the [plugin web site](https://techuptodate.com.au/what-if-bitcoin/).
